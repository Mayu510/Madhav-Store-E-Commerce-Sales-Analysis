# 📊 Madhav Store Sales Analysis & Dashboard (Power BI)

## 📌 Project Overview

This project focuses on analyzing sales data of an e-commerce store (Madhav Store) to uncover meaningful business insights.
The analysis is performed using **SQL and Power BI**, and the final output is an interactive dashboard.

---

## 🎯 Objectives

* Analyze sales and profit trends
* Identify top-performing customers and products
* Understand state-wise performance
* Evaluate different payment modes
* Detect profit and loss patterns

---

## 🛠️ Tools & Technologies Used

* **SQL** – Data extraction and querying
* **Power BI** – Dashboard creation and visualization
* **Excel** – Data preprocessing

---

## 📊 Dashboard Features

* KPI cards for **Total Quantity, Amount, and Profit**
* **Profit by Month** (trend analysis)
* **State-wise Profit Analysis**
* **Top Performing Customers**
* **Category & Sub-category Analysis**
* **Payment Mode Distribution**
* Interactive filters (Quarter & State)

---

## 🔍 Key Insights

* Overall profit shows fluctuations with some loss periods
* Highest profit observed in **November**
* Loss observed in months like **May and December**
* **Cash on Delivery (COD)** is the most used payment method
* Certain categories (like Clothing/Printers) contribute most to sales
* Top customers significantly impact revenue
